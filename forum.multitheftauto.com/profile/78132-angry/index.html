            1. VF # 33 Server blocks

              Angry replied to Angry's question in Client

              Hello everybody, First of all, the good news: The problem has been solved! Many thanks to @Dutchman101 . That was really a job removing everything. I uninstalled the plague called "cfosspeed" before, but the driver "cfosspeed6.sys" in the System32 folder was still there. As a user or administrator, I was unable to delete the file because another process is accessing it. Which process or service this is could not be seen from the error message. I then had to insert a backup USB stick and use it to start the command prompt. Since Windows did not allow me to access the C hard drive, I had to do a "workarround" using notepad.exe. Using the notepad in safe mode on the USB stick, I was finally able to delete the file "cfosspeed6.sys" in 'C:\Windows\system32' with administrator rights. That was a very long and hard road for such a problem. Everything for a program that I didn't want and didn't install but had problems with MTA. But now I can finally play again. I hope this post can help you too. The topic may can be closed. Friendly regards Angry
