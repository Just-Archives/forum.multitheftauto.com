An SDK is a software development kit, allowing you to create your own modifications for the software the SDK is for. The SDK is being developed for MTA by the team so we can make our own gamemodes for MTA. It will

be in one of the future releases when they got it ready.

The current MTA version and its core

I saw so many people asking to develop 0.5.x further and release it.
