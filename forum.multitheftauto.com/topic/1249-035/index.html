            On this beautifull sunday the team has had a chat session about the future of MTA. We decided to spend the next few weeks on bughunting and improvements to the current version. The next version of MTA will be 0.3.5 and will have the following features:

            * improved netcode (fixes lag problem and creates real sync)

            * all new GUI

            * less bugs

            * if possible : voice chat (depends on bandwith needs, maybe make it optional)

            Other improvements like game modes and chat may come in 0.3.5 but are not the top priority at this moment. If not included they will stay planned for 0.4 .
