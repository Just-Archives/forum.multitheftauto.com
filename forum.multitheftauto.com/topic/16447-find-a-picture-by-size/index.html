            I just randomly thought of this idea I dunno if its been done before.

            Rules/Steps

            1. Go to http://www.google.com/images and type "# bytes" to find whatever you need.

            2. Any images showing up are fair game

            3. Surround the URL with tags. Under the picture, specify the next number of bytes needed for the next picture. It should be typed as "# bytes" so it can be copied and pasted. Valid numbers: 10000-99999

            Other Notes:

            *If there is no pictures for the byte size type another number and post.

            *No dirty porno and such, read forum rules. You will be punished severely for something naughty

            So for example, heres a picture of 12310 bytes in size:

            64153 bytes
