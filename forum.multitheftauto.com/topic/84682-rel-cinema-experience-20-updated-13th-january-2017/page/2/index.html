            @kleinmarquez Ah, yeah. That's the downside of replacing objects in MTA. If you want you could change the objects ids though but you have to go and find an id that you want to replace. You can use the map editor to check the ids on objects in the world.

            Unpack the zip file and browse to the map folder and open TheCinema.map file with a text editor. Then replace the ids that you see in the first image.

            Then browse to the screen folder and open up screen.lua and change the ids to the ones you put in the in the map file. (see second image)
