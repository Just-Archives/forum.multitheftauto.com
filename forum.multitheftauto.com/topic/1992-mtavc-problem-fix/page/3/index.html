            I spent about 15min today reinstalling MTA:VC. Kept getting an error

            that it couldnt open files to update them.

            First try the 'uninstall' option in the mta setup, if this doesnt work (see below why not), then:

            Manually delete the current shortcuts and set any files in the /MTA/ directory to ARCHIVE (right click followed by properties, tickbox at the bottom) As it sounds as if they are there as READ ONLY.

            Then install once more.

            You MUST allow the installer to create the shortcuts and icons or it will NOT install in the correct place, and will therefore not work. You will also be unable to un-install
