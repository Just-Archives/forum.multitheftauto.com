            Notes - Rules

            Ok thanks to Juggernaut i'm setting up this first MTA:VC Problem fix.

            - Trainers conflict with MTA and probably mess up the whole game. We will try to disable these in further releases, please do not use them in combination with MTA:VC

            - Please see this topic before you're posting any problems with MTA:VC yourself!

            - Only post replies when you know how to solve problems you've experienced.

            (P* stands for Problem, A* stands for Answer)

            Problem Fixes

            P1: When i'm in a game, i can see the people but people can't see me!

            A1: Put MTA:VC on Windows 2000 compatibility mode and GTA: Vice City on Windows XP mode.

            P2: My game is crashing/giving unhandled exceptions

            A2: This happends sometimes.. if you're having a crash, you only have to restart Vice City and to start a new game, and you're back in. No need to restart MTA or your computer. If you're experiencing several crashes after each other, probably someone is messing up the game by using trainers or such. Enter another server and see if it's working now.

            Change your VC to english.

            If you still got problems, check your version settings in the MTA Client, if you've mods installed or using another program that modifies GTA in any way. It none of these work, which is very rare, try to reinstall Vice City and MTA and test it again.

            More problems will be posted

            P3: I keep shooting this guy but he never dies! I see blood all over but he still moving!

            A3: A good reason why this happends is because the player you're shooting at is either in the Vice City menu, so not syncing its data or alt+tabbed to Windows. Another reason is that you're playing with either a server that has not enough bandwidth to handle the amount of players connected (which requires alot of bandwitdth) or your connection can't handle the amount of data that must be uploaded (2 kb/s) or downloaded (+- 10kb/s with 15 players).

            P4: Also if you see the guy's guns lying around all of a sudden, but he's standing up, he is really dead and the camera is fading on his screen, but for some reason on other people's screen it doesn't show him lying on the ground.

            A4: The dead animation isn't synced very well. If the weapons are lying around him he's dead.
