            Multi Theft Auto'nun 1.1 versiyonu sizlere yeni özelliklerle tanıştıracak, Scripter'ların özenle hazırladığı yeni bir MTA

            Yenilikler listesi:

            MTA:SA Kod Değişikliklerİ.

            yeni anasayfa*:

            yeni gui çeşidi eklendi

            yeni başlıklar eklendi - 1.0.5 versiyonundada aktif

            serbest ekleyebilme olasılığı

            yeni sunucu listesi*

            Unicode kodlama desteği. 1.0 scriptlerinizi 1.1 UTF-8 BOM scriptlerine dönüştürebilirsiniz.

            performans değişimleri:

            MTASA programının başlama süresi azaltıldı.*

            FPS(Frame Per Second) limiti yükseltildi, 46 dan 70 e kadar çıkan FPS.

            yeni özellikler:

            added a possibility to change antialiasing and mipmapping setting within MTASA (?)

            Aero fix for Windows Vista/7 users - if you are bothered with the forced change of your Windows Aero scheme to Windows Basic everytime you launch MTASA, you can use an option in MTASA settings to make it stick with the Aero scheme(?)

            rasgele isim yaratıcı - 1.1 i yüklediğinizde size rasgele bir isim verilecek, böylece mta'daki başka oyuncuyla isminiz aynı olmayacak.

            linux sunucu şemalarında derlemeler yapıldı

            genel durum:

            uluslararası alt forumlar - aralık ayında açıldı

            steam grubumuzda 1200+ üye ( http://steamcommunity.com/groups/mta )

            120000'den fazla MTA Kullanıcısı

            yeni oyuncu rekoru - 4172 kullanıcı aynı zamanda oyundaydı ( 2011 May Sun 1st 17:00 UTC )

            Eğer yıldızlarla( * ) işaretlenmiş maddeler hakkında daha fazla bilgi öğrenmek için lütfen Video'yu izleyiniz (Yüksek Çözünürlük mevcut! )
