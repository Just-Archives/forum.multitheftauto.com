            Salvete,

            San Dierno is a german island city, which started in 2013. Now after 5 years the map will come back for a 7th Edition. It will be the last edition, where also a server will start soon.

            The Anniversary Edition is now in the Pre-Alpha Build 3. 

            German mainthread: https://www.mta-sa.org/thread/31309-5-years-san-dierno-anniversary-edition-the-last-and-7th-edition/?postID=430253#post430253

            How to find this map on MTA?
            IP:109.230.248.208:22003 or San Dierno in the search

            In this topic i post the changelogs from the island.

             

            Discord from San Dierno: https://discord.gg/Zer9GHX

            Salutem Dicit

            Sehr geehrte Damen und Herren,

            Wie gewünscht wird nun ein Around The Map präsentiert, wo man die Map in voller Größe betrachten kann. Selbstverständlich wurden bei diesem Around The Map wieder nur die äußersten Straßen abgefahren.
            Extra in diesem Video ist auch die Kartbahn, die mit der finalen Beta hinzugekommen ist, sowie mal ein Anspann, wenn man von San Dierno Beach geradeaus gefahren wäre zum Airport.
            @'MaxiR' wird demnächst in diesen Thread eine Server IP veröffentlichen, wo man die aktuelle Version 1.0.1a sehen kann.

            Auf eine Kritik würde ich mich persönlich sehr freuen, denn mit Version 1.0.1 ist diese Map noch lange nicht zuende.
