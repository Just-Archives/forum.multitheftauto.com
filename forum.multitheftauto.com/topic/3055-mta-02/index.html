            Q: Why does it say "Trainer usage suspected" again?

            A: Maybe you're using external programs that are trying to modify VC, such as the speedometer or whatever. Shut these down and try again.

            Q: I'm getting the your "VC files are wrong" thing and i haven't even installed any mods!

            A: You still have some modified files in your VC directory. If you've uninstalled any mods, that could be the problem. Even if there is 1 character changed in one of these files: handling.cfg, weapon.dat or mta.scm you'll not be able to connect. Reinstall VC and/or reinstall MTA.

            Q: I'm getting the normal single player when i start up Vice City!

            A: Please use the Start Game button in the MTA Client, this is the only way it works.

            Q: The Start Game button is not enabled! I can't click it!

            A: You have to be connected to a server before you can click the Start Game button to start your VC game.

            Q: It says i'm having a wrong version of VC or there's something else wrong with MTA and finding VC

            A: Try the following: go to your Windows directory, for example c:\windows, and remove the MTA.INI file. Then try to start the MTA client again.
