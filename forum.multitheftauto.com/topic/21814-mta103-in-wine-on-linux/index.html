            Try MTA 1.0.3 and latest Wine.

            Maybe, if you're using latest Wine (1.1.35) already, there might be a regression in it, as 1.1.34 has a good rating running MTA

            http://appdb.winehq.org/objectManager.p ... &iId=17751

            Although, there are very bad ratings with older Wine versions.

            So, try Wine 1.1.34.
