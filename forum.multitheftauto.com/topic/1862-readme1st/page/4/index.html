            As for the blue screen of death this is almost certain due to old/incorrect motherboard drivers. Never use the ones that come with XP as they dont work- d/l some from the motherboard manufacturers website.

            If the BSOD says something about a parity error then your memory is playing up- try pulling it out, blowing the dust off and then reseating it.

            The only other thing I can think of is that you need to install SP1a and the Quake 3 hotfix as this prevented a lot of my crashes.
