            Boredom caught up to me so I decided to make some minor aesthetic changes to the original MTA menu screen.

            MTA Screen Theme

            A very simple background that also functions as a logo, compiled with footage from my MTA server.

            SkyHigh Theme

            Replaced the original background screen grid with a panoramic image of the San Andreas map, and also applied the smaller version of the MTA logo for a less cluttered screen.

            IA Theme

            Aimed for the fanatics of Vocaloid, anime and JRPG's, or simply, Japanese players.

            What's an IA?

            Credits: I'd like to credit and thank forum user WolfPire for his assistance on spell-checking the theme menu!

            [/divbox]

            Sorry for the small images but apparently the MTA forums didn't apreciate the original resolution. There are however full resolution preview pictures inside each theme's .zip folder.

            How to install:

            [b]Drag and drop the files inside the 'files' zip folder into:[/b] 
[i]MTA San Andreas\MTA\cgui\images[/i]

            Feel free to mix themes if you like.
