            The game GTAIII itself you have to buy in a shop, or a online shop. You can't download it on the internet (not legally that is).

            About the new things, there are a lot of websites about these things. Just do a search in Google or any other search engine and you will find out.

            In the game GTAIII there is no multiplayer, MTA is a mod. A mod changes a game, in this case GTAIII is changed, multiplayer is added. You need the original to use it, MTA can be downloaded for free from this website.

            Currently the newest version is 0.2, it is a alpha version which means it is still under development and not yet finished. 0.3 is the next version, it should be much more stable and it will have more features, you can read about it on this website.
