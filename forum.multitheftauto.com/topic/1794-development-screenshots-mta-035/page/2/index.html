            Alright, to answer questions ....

            "can we ride the same car with other players?"

            Not currently ... there is code in place for this to work, but it has not been tested completely and is not working 100%. This will most likely not be included in 0.3.5 BETA, and possibly not the 0.3.5 FINAL, but something should be included in 0.4 if not a patch between the two.

            "so long as it notifies you when you've been disconnected"

            A message is displayed in game as well as the Client upon disconnection. This only really ever happens though when either your internet connection goes down or the Server's internet connections goes down (or Server application shut down).

            Internet Connections

            I am not 100% on the internet connections that were used in the movie, but I can tell you some of the internet connections used by the team members.

            I am currently on Cable (1500/384) and have absolutely no problem with lag. Sintax and AJH are on faster cable than myself and obviously also have no problems. trx is on dial-up and from what he has told me has very little / non-existent lag.

            We have programmed the Client / Server communication to be MUCH more efficient then any previous version. It was stated that the upload rate was some insane amount like 16 kb/s (am I correct?) on the old system. Well, the new communication standard does not exceed 2-3kb/s in theoretical terms (untested). All in all, we haven't had any problems and further optimizations can be done.
