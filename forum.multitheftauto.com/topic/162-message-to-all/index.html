            First of all i wonna thank everyone in the name of the team for the support you al give us. Without the community the development wouldn't be that much fun.

            When we decided to release our Alpha and beta versions we did it because we wanted to prove that it works.Now these days we sometimes regret that decision. When we read the forum or we are chatting on our IRC channel there's always someone asking when the next version will be released, what is changed and so on. Therefore we request not to post these questions anymore we had to lock several topics about it already on the forum. You can find all information we have ourselves on the forum in the anouncements under "General MTA Discussion"

            When you start a new topic, please make sure the answer isn't already on the site or the forum. The forum has a search function. Try it first. We have found multiple topics that handle about the same thing. This means we have more work moderating and reading everything because we have to read things twice!

            When you reply to a topic make sure you kniw what it is about and that you have read everything. A good example can be found in the anouncement of the delay. We find there the question about what a shuttle pc is. That question wasn't asked for the first time so the next posts says that you have to read the topic again and you will find the answer. The next post asks the same question again. That isn't normal. If you read a topic read it completely and not only the first 1 or 2 posts!

            Spamming of the forum will not be allowed. We are gonna start a blacklist so we can monitor the people suspected of spamming a bit more. If we see people spamming after warnings we wil use the simplest solution: reset there number of posts because thats what spamming is all about isn't it? the most posts! I just have to go in the db and reset that number to 0 !

            The speed problem of the site and forum aren't gonna be solved over 1 night but the biggest part of the problem should be solved within one or two weeks. Our host BASM has bought a complete rack @ IO.NL and is now looking for the correct materials they still need. The servers will be moved to there new location as soon as it is ready. This doesn't solve the speed problem because it is rtelated to server load, but together with the move a new server wil be installed. And another server is planned and will be installed before july.

            We hope our message has been clear and hope we can keep getting the support of everyone!

            The MTA team and forum moderators
